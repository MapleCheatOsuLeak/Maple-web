Relax

Maple будет нажимать на кружочки за вас. Дайте вашим пальцам немного отдохнуть.

Aim Assist*

Помогает наводиться на кружочки. Теперь вы уж точно сможете фкшнуть ту самую мапу Сотаркса.

Timewarp

Играйте на любой скорости.

Replay Bot*

Совсем устали играть? С Replay Bot вы сможете скопировать любой реплей. Просто расслабьтесь и наслаждайтесь.

Visuals

Включает в себя AR Changer, HD/FL Removers и настройку пользовательского интерфейса.

Spoofer*

Позволяет создавать и играть на нескольких osu! аккаунтах.

Misc

Включает в себя no spectator mode, score submission prompt и Discord RPC spoofer.

* - Это функции могут быть удалены из Maple Lite при выходе Maple Full.
